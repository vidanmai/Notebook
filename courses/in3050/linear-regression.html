What is Linear Regression?

The objective of a linear regression model is to find a relationship between one or more features (independent variables) and a continous target varaible (dependent variable). We need some idea regarding the kind of functions we are looking for. The simplest is to assume a linear function:

When there is only one input variable, it is called simple linear regression:
- f(x_1) = w_0 + w_1 * x_1
- A straight line
- Easy to draw
In the general case:
f(x) = w_0 + w_1 * x_1 + w_2 * x_2 + ... w_m * x_m

Training a Linear Regression Model

Training of the model here means to find the parameters so that the model best fits the data.

How do we determine the best fit line?

The line for which the error between the predicted values and the observed values is called the best fit line or the regression line. These errors are also called residuals. The Residuals can be visualized by the vertical lines from the observed data value to the regression line.

When you want to build a model(linear regression model), you want to use the least-squares error method that is minimizing the "total" euclidean distance between a line and the data points. Theoretically the coefficients of this line can be found using calculus but in practice, algorithm will perform a gradient descent which is faster.

Once you have your model, you want to evaluate its performances. In the case of regression, it would be good to compute a metric that evaluates "how far" the model is to the actual data points (or test set data) in average. The MSE (Mean Squared Error) is a good estimate for this.
In Summary, the LSE is a method that builds a model and MSE is a metric that evaluates your model's performances.

Mean Square Error (MSE)

We need a way to tell whether a line is a good fit to the data, and whether one line is better than another. The goal is to minimize the error.

  • The purple dots are the points on the graph. Each point as an x-coordinate and a y-coordinate.
  • The blue line is our prediction line. This is a line that passes through all the points and fits them in the best way. This line contains the predicted points.
  • The red line between each purple point and the prediction line are the errors. Each error is the distance from the point to its predicted point.

Gradient Descent

We minimized J(θ) by trial and error above - just trying lots of values and visually inspecting the resulting graph. Surely, there must be a better way? This is where Gradient Descent comes in to play. Gradient Descent is a general function for minimizing a function, in this case the Mean Squared Error cost function.

Gradient Descent basically, just does what we were doing by hand - changing the θ values, or parameters, bit by bit, until we hopefully arrived at a minimum.

We start by initializing θ_0 and θ_1 to any two values, say 0 for both, and go from there. Formally the algorithm is as follows:
